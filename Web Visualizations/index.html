Summary: Weather Plots



Visualizations



In this project students attempted to create visualizations using Matplotlib of weather patterns close to the equator. These visuaizations compare latitude with a variety of parameters incluing Temperature, Humidity, Cloudiness and Wind Speed in both the Northern and Southern Hemispheres. Using ideal values of each of these parameters, analysis was done to find the best vacation spots.
